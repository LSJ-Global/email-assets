F5 AppWorld Seoul 2026 EDM - OFT conversion guide

Quick start for non-technical users
1. Keep this folder structure intact, including the assets folder.
2. On a Windows PC with Outlook installed, double-click Run_Create_OFT.bat.
3. Wait until the script finishes.
4. Check the same folder for F5_AppWorld_Seoul_2026_EDM.oft.
5. Open the .oft file in Outlook and verify the images and button link.

If double-clicking the BAT file does not work
1. Close Outlook first and try again.
2. Ask IT whether PowerShell script execution is blocked on your PC.
3. If needed, run this command in PowerShell from this folder:
   powershell -ExecutionPolicy Bypass -File .\Create_OFT_Windows_Outlook.ps1

Alternative manual workflow
1. Open F5_AppWorld_Seoul_2026_EDM.eml in Outlook.
2. Confirm the image body and the F5 registration button link render correctly.
3. Choose File > Save As.
4. Set Save as type to Outlook Template (*.oft).
5. Save as F5_AppWorld_Seoul_2026_EDM.oft.

Important for real campaigns
- Replace local image paths or CID images with the sending platform's hosted HTTPS image URLs if your ESP does not preserve inline CID images.
- Add unsubscribe and sender postal address in the ESP/template footer for compliance.
